#### Installation 
1. Clone the repo
```sh
git clone https://github.com/vidhik2002/Canvas.git
```
2. Move into folder
```sh
cd Terminal
```
3. To install latest version of npm
```sh
npm install -g npm@latest
```
4. Add .env file
```sh
PORT = <YOUR_PORT>
```
5. Run the commands
```sh
npm i express
nodemon index.js
